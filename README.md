# Task 1 - Data Immersion & Wrangling

## Project Overview

This project was completed as part of the ApexPlanet Data Analytics Internship Program.

The objective was to perform data cleaning and preprocessing on a housing dataset to transform raw data into an analysis-ready format.

---

## Objectives

* Create and understand dataset structure
* Identify missing values
* Remove duplicate records
* Handle inconsistent data
* Treat outliers
* Standardize column names
* Perform feature engineering
* Export cleaned dataset

---

## Tools & Technologies Used

* Python
* Pandas
* NumPy

---

## Files Included

* `housing_data_creator.py` – Generates sample housing dataset
* `data_cleaning_project.py` – Cleans and preprocesses the dataset
* `housing_data.csv` – Raw dataset
* `cleaned_housing_data.csv` – Final cleaned dataset

---

## Key Data Cleaning Steps

### 1. Data Loading

Loaded raw housing dataset into Pandas DataFrame.

### 2. Missing Value Handling

* Numerical columns filled using median
* Categorical columns filled using mode

### 3. Duplicate Removal

Removed duplicate rows for data consistency.

### 4. Outlier Treatment

Applied IQR method to detect and cap outliers.

### 5. Feature Engineering

Created new column:

* `house_age`

### 6. Export

Saved cleaned dataset for future analysis.
